### What we need to show

The events `goodBlockEvent L D c rate k n j` (indexed by `j : Fin k`) are mutually independent under `D^{k*n}`.

### Key insight

Each `goodBlockEvent j` is the preimage of a **fixed** measurable set `A` under the **j-th** block extraction function. The block extraction functions are already known to be `iIndepFun` (from `iIndepFun_block_extract`). Independent functions preimaged against measurable sets give independent sets.

### Route

**Step 2**: Convert `iIndepFun` to `iIndepSet` via the sigma-algebra route.

- `ProbabilityTheory.iIndep_of_iIndep_of_le` bridges: if sigma-algebras of the sets are ≤ sigma-algebras of the functions, we're done
